# Art Code Landing

Art Code Landing is a website showcasing course for practicing creative coding skills. Created during ArtMasters Academy course "Из мастеров в наставники".

## Description

![landing demo](./source/demo.gif/)

Landing page is a description of programming courses for designers and those who are not familiar with programming. 
On the landing page there is brief information about the course, description of learning outcomes, target audience, learning process, brief course content, examples of works and brief information about the author.

Компьютерная графика, управляемая кодом на языке программирования Processing.

О курсе

Курс «Основы программирования Processing для дизайнеров» дает возможность изучить программирование генеративной компьютерной графики и углубиться в техническую сторону создания интерактивных приложений.

Генеративное программирование позволяет компьютеру автоматически создавать художественные работы на основе определенных человеком алгоритмов. Такое креативное программирование использует программный код для создания компьютерной графики, с акцентом на доступном языке программирования Processing, специально созданном для дизайнеров.

Результаты обучения

По завершении курса слушатель сможет:

  • Применять креативное программирование в своих проектах для достижения желаемых результатов.
  • Создавать интерактивную графику и отлаживать программный код.
  • Понимать структуру компьютерных программ.
  • Применять алгоритмические конструкции.
  • Работать в среде программирования.

Для кого этот курс?

Курс отлично подойдёт тем, кто:

  • Занимается графическим или веб-дизайном и хочет применять в своих проектах генеративную программируемую графику.
  • Хочет открыть для себя новую сторону компьютерной графики, которая позволяет создавать уникальные проекты.
  • Совершенно не разбирается в программировании, но очень хочет научиться.

Программа курса

36 часов • 11 уроков • 6 заданий • 5 тестов

Основы программирования графики

  • Создание и настройка геометрических форм с контролем заливки, обводки и прозрачности, а также использование различных режимов рисования и пользовательских фигур.
  • Тест «Основы программирования графики».

Программируемая типографика

  • Создание шрифтов и вывод текста, безусловные циклы, типы данных, условные операторы, преобразование объектов, использование массивов.
  • Тест «Генеративная типографика».

Интерактивный дизайн

  • Отслеживание действий пользователя в режиме реального времени.
  • Тест «Интерактивный дизайн».

Итоговый проект

  • Использование полученных навыков для создания генеративного дизайна на основе цикла, геометрических преобразований и текста.
